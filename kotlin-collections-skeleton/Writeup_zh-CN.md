# Kotlin: 数据结构

> 难度评价：4/5

我对这卷的评价就是难搞。真难搞。

1. 首先，这份试卷要求一定要写 SinglyLinkedList 的额外测试点。要想得出来哪些测试点需要更新，程序里的那些运行逻辑需要测试，就比较复杂。到最后才占 10 分，这部分所需的精力与其占分并不匹配。
2. 第二题还好，难度不大。可以直接按照试卷 spec 指示写完类型定义就完事了。
3. 同上，removeAll 这题直接照 spec 写代码，秒杀。
4. 需要熟悉 Java 语言中的各种特性。这也就是语法转换的题目，熟悉 Java 同样可以轻松做完。
5. 这题全卷最难。每次调用函数之前得先获取互斥锁，然后再释放。但获取和释放还得顾及其他线程会不会对对象造成更改，做起来很麻烦。

> 综合评价：这卷可以作为挑战来答，虽说做着要比其他卷子要麻烦，总体来说是一份很好的锻炼。推荐最后做。
